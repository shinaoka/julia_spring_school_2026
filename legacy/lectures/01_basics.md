### 1. REPLの基本（10分）

- **モード**:
  - `julia>`: 通常
  - `]` : Pkgモード（`add`, `st`, `test` など）
  - `?` : ヘルプモード
  - `;` : シェルモード

よく使うもの:

```julia
versioninfo()
ans
```

### 2. 変数・型・数値（10分）

```julia
x = 1
y = 1.0
typeof(x), typeof(y)
```

- **動的型付け**: 値に型がある（変数に固定型があるわけではない）
- ただし **型は非常に重要**（性能・コンパイルに直結）

### 3. 関数（15分）

```julia
f(x) = x^2 + 1
f(3)
```

複数メソッド（多重ディスパッチ）:

```julia
g(x::Int) = x + 1
g(x::AbstractFloat) = x + 0.5
g(1), g(1.0)
```

### 4. 配列とブロードキャスト（20分）

```julia
a = [1, 2, 3]
b = rand(3)
a .+ b
sin.(b)
```

- `.` は「要素ごと」を表す（ブロードキャスト）
- `@.` マクロで式全体に付けられる

スライスとビュー:

```julia
A = rand(5, 5)
col = A[:, 1]      # これはコピー（デフォルト）
vcol = @view A[:, 1]  # これはビュー（参照）
```

### 5. 制御構文（10分）

```julia
s = 0.0
for i in 1:10
    s += 1 / i
end
s
```

内包表記:

```julia
xs = [i^2 for i in 1:10]
```
